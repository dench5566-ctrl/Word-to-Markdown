# d2 — DOCX → Markdown

Простое и быстрое мобильное приложение для конвертации `.docx` файлов в Markdown **полностью оффлайн**.

## Особенности

- Полностью работает без интернета
- Одна большая кнопка — минимум действий
- Чистый и читаемый Markdown на выходе
- Сохранение и шаринг результата
- Apple-style минималистичный дизайн
- Тёмная тема поддерживается

## Стек

- Flutter 3.24+
- Riverpod
- `docx_to_markdown` пакет
- Чистая архитектура + конфигурируемые JSON

## Запуск

```bash
flutter pub get
flutter run
```
